### My First React App Project :)

###### Technologies

- HTML,Css
- ES6
- React
- custome Hooks
- useState & useEffects Hooks
- Json Server

###### Features

- Showing Blogs
- Showing Blog Details
- Adding New Blogs
- Deleting Blogs

## Installing / Getting started

```shell
npm install

npm run start

npx json-server --watch data/db.json --port 8000
```
